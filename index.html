# Membuat File Path

File Path
# Path Deskripsi
1 src="picture.jpg" File "picture.jpg" terletak di folder yang sama dengan halaman saat ini
2 src="images/picture.jpg" File "picture.jpg" terletak di folder gambar di folder saat ini
3 src="/images/picture.jpg" File "picture.jpg" terletak di folder gambar di root web saat ini
4 src="../picture.jpg" File "picture.jpg" terletak di folder satu tingkat di atas folder saat ini
